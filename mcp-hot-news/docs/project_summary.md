# mcp-hot-news 项目总结

## 1. 项目概述

mcp-hot-news是一个基于MCP (Model Context Protocol)的新闻热点服务，专门为AI Agent提供实时新闻热点数据。该项目从多个新闻源（微信、知乎、微博）获取数据，经过清洗、分类和聚合后，通过MCP协议暴露给AI Agent使用。

## 5. 使用示例

### 5.1 启动服务

```bash
python -m src.server
```

### 5.2 MCP工具调用示例

```python
# 获取热点新闻
get_hot_news(limit=10)

# 按新闻源获取新闻
get_news_by_source(source="微信", limit=10)

# 按分类获取新闻
get_news_by_category(category="科技", limit=10)

# 搜索新闻
search_news(keywords="人工智能", limit=10)
```
