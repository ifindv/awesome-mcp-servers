# mcp-hot-news 系统设计文档

## 1. 系统概述

mcp-hot-news是一个基于MCP (Model Context Protocol)的新闻热点服务，专门为AI Agent提供实时新闻热点数据。该服务将从多个新闻源获取数据，经过清洗、分类和聚合后，通过MCP协议暴露给AI Agent使用。

### 2.2 组件说明

#### 2.2.1 MCP Server层
- **Tool Registry**: 注册和管理所有新闻相关工具
- **Request Handler**: 处理来自AI Agent的请求

#### 2.2.2 News Service层
- **News Fetcher**: 负责从各新闻源获取原始数据
- **News Processor**: 负责数据的清洗、去重、分类和格式化

#### 2.2.3 News Provider层
- **WechatProvider**: 微信热文数据提供者
- **ZhihuProvider**: 知乎热榜数据提供者
- **WeiboProvider**: 微博热搜数据提供者
- **CustomProvider**: 自定义新闻源提供者接口

## 3. 核心功能

### 3.1 获取今日热点新闻
- 从所有配置的新闻源获取数据
- 按热度排序返回
- 支持限制返回数量

### 3.2 按分类获取新闻
- 支持科技、娱乐、体育等分类
- 可自定义分类规则
- 支持多分类查询

### 3.3 获取新闻详情
- 根据新闻ID获取完整内容
- 包含元数据和统计信息

### 3.4 关键词搜索新闻
- 支持模糊搜索
- 支持多关键词组合
- 返回相关性排序结果

### 3.5 获取新闻趋势分析
- 提供新闻热度趋势
- 支持时间范围筛选
- 可视化数据支持

## 5. 数据流设计

### 5.1 数据获取流程

```
AI Agent Request
    ↓
MCP Server接收请求
    ↓
News Fetcher从各Provider获取数据
    ↓
News Processor处理数据
    ↓
返回格式化结果给AI Agent
```

### 5.2 数据处理流程

```
原始数据
    ↓
数据清洗
    ↓
数据去重
    ↓
数据分类
    ↓
数据排序
    ↓
格式化输出
```
